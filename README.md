## Quick Start

Requirements:

- Node.js LTS
- npm

Run locally:

```bash
npm install
npm run dev
```

Run with Docker:

```bash
docker compose up --build
```

Then open:

- `http://localhost:8080`

## Quality Checks

```bash
npm run test:run
npm run build
```
